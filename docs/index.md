## üìñ What is PySophosCentralApi?

PySophosCentralApi is a professional-grade Python library and CLI application for interacting with Sophos Central APIs. It provides:

- **Complete API Coverage**: Full support for Sophos Central Endpoint API and Common API v1
- **Dual Interface**: Use as a library (async/sync) or CLI tool
- **Rich Features**: Filtering, pagination, export, and more
- **Type-Safe**: Full type hints for better IDE support
- **Well-Tested**: 219+ tests with 72% code coverage

## üéØ Key Features

### For Library Users
- **Async-First Design**: Built with modern async Python for optimal performance
- **Synchronous Option**: Sync wrappers available for easy integration
- **Type Safety**: Complete type hints throughout
- **Pydantic Models**: Type-safe data models with validation
- **Advanced Filtering**: Powerful query builders for complex queries
- **Export Capabilities**: JSON and CSV export with customization

### For CLI Users
- **Intuitive Commands**: Easy-to-use command structure
- **Multiple Output Formats**: Table, JSON, CSV
- **Rich UI**: Colored output with tables and progress bars
- **Configuration Management**: Easy setup with TOML files or environment variables
- **Batch Operations**: Process multiple items efficiently
